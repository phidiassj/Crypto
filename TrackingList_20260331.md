## 本輪優先觀察

### 相對較值得短中期關注

1. **ASTER**
   近期事件線最完整，Aster Code 與 Genius Terminal 都是直接指向交易量與生態擴張的催化；缺點是波動與敘事交易成分高。
2. **MNT**
   AUM、TVL、Bybit 生態導流三者同時成立，基本面比單純平台幣更厚，若市場風險情緒回暖，延續性可能優於多數中型幣。
3. **LINK**
   採用面比價格面更強，X Layer 與 OnePay 都是實際分發渠道；若 BTC 穩住，LINK 較有機會成為資金回流的高品質標的。
4. **CAKE**
   通縮與產品迭代仍在，若 DEX 敘事續強，CAKE 是低 RSI 且基本面仍持續更新的老牌選項。

### 可以觀察但不宜高追

- **GT / KCS / OKB / MX**
  都有平台層面的營運更新，但驅動大多是活動、上幣、API 或儲備金，較像交易所 beta，爆發力不如明確新產品敘事。
- **ZEN / NEXO / QNT**
  各自有監管、機構、企業採用題材，但價格反應通常偏慢，適合等結構更完整再處理。

### 偏保守看待

- **GAS / CVX / DASH**
  雖然 RSI 不高，但近期催化不足或帶有歷史包袱，較適合反彈交易，不適合直接當中期主軸。

## 方法與限制

- CoinMarketCap RSI 頁面今天仍只穩定暴露 100 筆可視資料，因此**目標清單2 不是對 60 檔候選的完整 RSI 普查**，而是對可視資料內可交叉出的結果排序。
- CoinGlass 公開端不適合批量逐幣穩定抓精確 liquidation map，因此本報告的清算帶屬**推估區間**，主要用於規劃風險，不應視為精準掛單位。
- QNT、DASH 這類標的在快速反應站上的直接新訊較少，這種情況本身就代表「缺乏新催化」。
