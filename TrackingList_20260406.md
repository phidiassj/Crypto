## 今日分組觀察

### 1. 可優先追蹤

- `BTC / XRP / LINK / BNB`：都有比單純技術反彈更完整的制度、資本流或基礎設施擴張理由。若市場沒有新的宏觀利空，這組最值得保留在核心追蹤名單。
- `ENS / FIL / LTC / XMR / DCR`：屬於品質較乾淨的第二梯隊。雖然爆發力未必最強，但中期敘事相對可講，適合等關鍵支撐確認。

### 2. 偏平台或高 beta 事件交易

- `GT / KCS / HYPE / LIT / SOL`：都有平台內部效用、流量入口或結構性故事，但這輪 `refx` 很清楚地告訴我們市場正在重估交易與 DeFi infra 風險，因此它們的彈性與波動會一起放大。
- `HYPE / SOL / LIT` 的事件線最強，但也是最容易被安全、流量轉化或風險偏好快速翻轉影響的一組。

### 3. 先保守看待

- `CVX`：賣壓與 fraud overhang 尚未清掉，暫時不把它視為中期配置。
- `GT / KCS` 雖然不差，但在 wallet / self-custody 敘事變強的背景下，平台幣暫時較像防守型輪動，不像新主線。
