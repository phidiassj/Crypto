# 每週加密貨幣關注重點 2026-05-03

> 整理區間：2026-04-26 至 2026-05-03 每日追蹤報告
>
> 下週觀察區間：2026-05-03 至 2026-05-10

## 一句話結論

下週主軸應收斂成「優先看 `BGB / NEXO / OKB / APT` 這組支付 rails、交易所分發、合規抵押品與高性能基建；其次看 `TAO / LPT / ZEN / BCH / XAUt / PAXG` 這類有 AI、隱私運算、支付升級或防守型 RWA 支撐的中期題材；`ASTER / HYPE / ORDI / GENIUS / LIT` 這組則只適合當高 beta 事件交易，不應預設成核心配置」。本週日報反覆出現的共識很一致：`油價 / 伊朗 / 荷姆茲` 讓市場風險偏好無法全面擴散，但 `BTC ETF / 穩定幣市值 / 合規借貸 / 交易所支付基建` 仍持續提供資金承接，因此盤面更像結構性選股，而不是全面 altseason。

## 下週核心觀察名單

### 1. 支付 rails、交易所分發與合規金融入口

- `BGB`
  - 這週從 `Morph $150M payment accelerator`、`Cobo` 導入，到 `VIP Fast Track` 與交易獎勵，幾乎每天都在強化同一件事：BGB 不只是平台幣，而是 `支付 + 分發 + 合規入口` 的混合資產。
  - 下週關鍵是 `1.92-2.00` 區間能否持續承接。若穩站 `2.00` 上方，BGB 仍是這輪中大型幣裡最乾淨的支付/平台主線代表之一。
- `NEXO`
  - 本週後半段穩定進入前排，原因非常一致：`美國合規回歸`、`新增 SOL / XRP 抵押品`、`阿根廷與歐洲數位金融合作`，都把它推向 `CeFi 收益 + 合規借貸 + 支付資金效率` 這條中期主線。
  - 下週觀察 `0.87-0.90` 是否能持續守穩。若市場風險偏好再度轉保守，NEXO 這種有現金流與制度化敘事的標的，通常比高 beta 題材更有機會留在核心名單。
- `OKB`
  - 本週最重要的新線索，是 `OKX` 將 `BlackRock` 代幣化美債抵押品接入保證金系統，讓 OKB 不再只是交易所效用幣，而是直接連到 `tokenized collateral + exchange compliance`。
  - 它的問題不是敘事不夠，而是代幣價值捕捉可能慢於消息速度。下週若市場持續交易合規抵押品與制度化衍生品，OKB 仍應維持高順位，但不宜把它當低位追價標的。
- `APT`
  - `Confidential APT`、`post-quantum / commodity` 敘事、`高吞吐與支付擴張`，讓 APT 成為本週反覆被抬高排序的高性能鏈代表。
  - 下週先看 `0.94-1.00` 區間。若站回 `1.00` 並守穩，它會比多數舊公鏈更像結構修復，而不是單純反彈。

### 2. 有基本面支撐的 AI 與新型基建主線

- `TAO`
  - 這週幾乎整段都維持在高優先級，核心邏輯沒有變：`AI infra`、`真實收入`、`Nvidia / Polychain / Grayscale / ETF` 類機構背書，讓它仍是 AI 題材裡最像中期核心資產的名字。
  - 但 TAO 已經從「可承接」轉成「追趨勢」。下週更適合等 `270-255` 區間確認支撐，而不是把它當成便宜的 AI 補漲標的。
- `LPT`
  - `5/31 Gateway` 升級、支付 SDK 與 AI video infrastructure 讓它本週雖然不算最熱，卻一直維持在質地較乾淨的基建名單中。
  - 如果下週 AI 題材繼續擴散，LPT 會比純情緒型 AI 幣更值得保留；但若沒有量能跟進，它仍偏向中段觀察而非前排主攻。
- `ZEN`
  - 本週後段開始明顯升溫，因為市場已不再只把它當舊隱私幣，而是看到 `Base 上的隱私 Layer 3`、`confidential compute`、`SoDex TGE` 與 staking relaunch。
  - 若 `5.9-6.2` 區間能守住，ZEN 會是下週值得保留的次核心題材；但若資金重新只追支付與大型交易所基建，它的排序仍應落後於 BGB / NEXO / OKB。
- `BCH`
  - `Kraken Institutional custody` 加上 `5/15 Layla upgrade`，讓 BCH 這週不只是老牌支付鏈，而是有明確時間表的技術事件交易。
  - 但 BCH 節奏仍偏慢。下週只有在 `470` 一帶被有效突破時，才值得往上提高優先級。

### 3. 防守型 RWA 與黃金代理

- `XAUt / PAXG`
  - `5/1` 的日報很清楚地提醒了一件事：當 `油價 / 伊朗 / 荷姆茲` headline 反覆主導時，代幣化黃金是少數能同時接住 `risk-off + RWA + 交易所流動性` 的資產。
  - 它們不一定是下週報酬彈性最大的標的，但若宏觀風險升溫，反而會是比大多數 alt 更合理的防守觀察組。

## 次核心觀察名單

### 1. 值得跟蹤，但還缺一層確認

- `MORPHO / PENDLE / LINK`
  - 這組共同特徵是基本面都很完整，但本週沒有像 `BGB / NEXO / TAO` 那樣反覆佔據日報最前排。
  - `MORPHO` 吃得到機構借貸、Coinbase backend、固定利率與 RWA；`PENDLE` 吃得到 `real yield / rate market / tokenization`；`LINK` 則仍是最乾淨的 `enterprise oracle + tokenization` 標的之一。
  - 問題不在質地，而在本週市場的短線偏好明顯更先給支付、交易所與 CeFi 借貸，所以它們比較適合列在第二層觀察名單。
- `MNT / MX / KCS`
  - 這組都和平台、資產負債表或支付延伸有關，但敘事密度與新增量催化都略弱於 BGB / OKB。
  - `MNT` 有 treasury 與 DeFi 基建承接；`MX` 有 PoR 與通縮；`KCS` 有支付卡擴張。
  - 下週若市場更願意回到平台幣補漲，這組有升級空間；否則先維持中段排序。

### 2. 有題材，但更偏輪動或事件交易

- `HYPE / ASTER / ORDI / LIT`
  - 這組共同特徵是都有真實產品或高活躍度支撐，但波動明顯更大。
  - `HYPE / ASTER` 是 `on-chain perps infra` 與交易量的代表；`ORDI` 是 `Bitcoin beta`；`LIT` 則偏 `RWA + perps infra` 的高 beta 交易票。
  - 下週若市場風險偏好延續，它們會有很高彈性；但一旦宏觀 headline 轉差，通常也是最先被去槓桿的一組。

## 下週最好先降低優先級

- `GENIUS`
  - 本週量能與話題都很強，但主體仍是新幣、永續上線與同名監管詞干擾造成的高波動交易。
  - 若沒有新的實質產品與持續流動性支持，下週更適合當短打觀察，而不是中期研究主軸。
- `RIVER / SIREN / BEAT`
  - 這組是本週最一致的高風險名單：`供給集中 / 操縱疑慮 / 下架風險 / 薄流動性` 反覆出現。
  - 即使 RSI 偶爾偏低，也不應把它們當成安全超跌名單。
- `IP / AR / DASH / DCR / BSV`
  - 不是完全不能追蹤，而是本週證據顯示它們都缺少足以把排序拉到前排的新催化。
  - 在下週仍有 `油價 / 地緣風險 / 穩定幣與制度化資金偏好` 的背景下，這類題材通常會被更清楚的支付、借貸與交易所主線壓住。

## 本週整合出的市場背景

### 1. 盤面仍是結構性 risk-on，不是全面 altseason

- 4/26 到 5/3 的日報非常一致：`BTC ETF / 合規借貸 / 穩定幣市值 / 交易所支付基建` 持續提供支撐，但這股資金還沒有強到全面擴散到所有山寨幣。
- 所以下週不應用普漲心態配置，而是持續集中在 `支付 / 交易所 / 合規借貸 / AI 龍頭 / 少數可驗證基建資產`。

### 2. 最乾淨的主線仍是支付、分發、抵押品與合規金融基建

- `BGB / NEXO / OKB / APT` 反覆出現，不是因為它們漲得最瘋，而是市場目前最願意給估值的方向，就是 `支付 rails、交易所分發、可用抵押品、合規借貸與高性能結算基建`。
- 換句話說，下週應優先追那些能承接穩定幣流量、制度化交易需求或有實際產品導流的資產，而不是只靠短期人氣的高 beta 幣。

### 3. AI、perps 與 BTC beta 可以做，但要嚴格分成核心與交易倉

- `TAO / LPT / ZEN` 這組仍屬值得保留的中期題材，因為都有 `收入、升級、機構背書或協議轉型` 的支撐。
- `ASTER / HYPE / ORDI / GENIUS / LIT` 則比較像交易倉，因為它們雖然有量能與題材，卻更依賴風險偏好、槓桿情緒與事件熱度。
- 這個區分下週很重要，否則容易把高 beta 的短打標的誤放進中期配置。

### 4. 宏觀與地緣風險仍在，防守組不能完全移除

- 本週多次提到 `伊朗 / 荷姆茲 / 油價`，代表外部風險沒有消失。
- 因此 `XAUt / PAXG` 這類代幣化黃金雖然不是主流 alt 交易標的，卻是下週做防守輪替時最值得保留的觀察組。

## 建議的下週觀察順序

1. `BGB / NEXO / OKB / APT`：先看支付、交易所分發、合規借貸與抵押品主線是否延續。
2. `TAO / LPT / ZEN / BCH`：確認 AI、隱私運算與支付升級題材是否持續得到資金承接。
3. `XAUt / PAXG`：若宏觀 headline 轉差，優先檢查防守型 RWA 是否重新吸金。
4. `MORPHO / PENDLE / LINK / MNT / MX / KCS`：列為第二層研究與輪動追蹤名單。
5. `ASTER / HYPE / ORDI / LIT / GENIUS`：維持高 beta 事件交易定位，不列核心配置。
6. `RIVER / SIREN / BEAT / IP / AR / DASH / DCR / BSV`：只做風險觀察，不列優先研究清單。
