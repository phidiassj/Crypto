## 重磅關注

1. `BTC` 今天最明顯的不是新敘事，而是 **資金面與需求指標同步轉弱**。`refx` 最後兩份都提到 `BTC ETF` 連日淨流出、`BlackRock` 與遠古巨鯨轉帳壓力、以及需求指標滑到今年低點；`BlockTempo` 也直接把「連 6 日失血 `15.5 億美元`」寫成主標。這代表 `BTC` 仍是整個市場的流動性錨，但短線不再是最乾淨的進攻資產。
2. `ETH / RWA / stablecoin / tokenized distribution` 仍是今天最完整的制度化主線。`crypto.news` 補上 `RWA` 規模已逼近 `340 億美元`、`Prometheum` 認為鏈上證券卡在分發、`MoonPay` 把買幣入口塞進 `ChatGPT`；`BlockTempo` 與 `CoinGecko News` 又同步補到 `Tether x Georgia GELT`、`Coinbase` 的 `8` 大金融升級方向、`CLARITY` 與穩定幣風險管理。資金不是離開 crypto，而是更集中往 `可分發、可合規、可支付、可鏈上包裝` 的軌道移動。
3. `Hyperliquid / prediction markets / 鏈上跨資產交易` 正從高 beta 故事走向基建故事。`refx` 與 `crypto.news` 都提到 `Hyperliquid` 往 `CPI prediction market`、`HIP-4 outcome contracts` 推進，前一晚的 X 也把它跟 `RWA perps / pre-IPO / prediction` 串在一起。這表示資金追的已不只是 `HYPE` 價格，而是「鏈上交易所能不能吃下更多非純幣圈資產」。
4. `美伊 / Hormuz` 仍是全市場最大的外部變數，但今天更像 **headline de-risk**，不是風險真正結束。四份 `refx` 都圍繞 `談判推進 -> 油價走弱 -> 風險偏好回升`，同時又保留 `核材料、Hormuz 重開、能源供應恢復速度` 仍未完全落地。這條線直接決定 `BTC` 能不能止跌、`alt` 的 risk-on 能不能延續。
5. `AI / quantum / cross-chain compliance / 政治金流` 是今天浮出水面的下一層基建過濾器。`crypto.news` 一邊寫 `Bitcoin` 的 post-quantum transition 不能倉促上路，一邊寫 `cross-chain AML gap`、`Fairshake` 對德州選戰砸下 `500 萬美元`；`CoinGecko News` 又把 `Kohaku` 錢包隱私 SDK、`CLARITY`、`Coinbase` 對穩定幣風險治理的說法補齊。這些訊號說明市場下一輪不只比誰發幣或上幣，而是比誰更能穿過 `監管 / 安全 / 合規` 的門檻。
