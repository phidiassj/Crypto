## 今日分組觀察

### 1. 可優先追蹤

- `BTC / ETH / LINK / BNB / MORPHO`：有比單純技術反彈更完整的機構、基礎設施或真實產品敘事，且在 `refx` 的主線下也較能承接穩定幣、合規或資本流方向。
- `LINK / MORPHO` 是今天最像「若市場回暖會優先吃到資金」的兩檔中大型基礎設施標的。

### 2. 第二梯隊觀察

- `FIL / NEAR / CAKE / KCS`：結構不差，也有各自敘事，但都還缺少單一足夠強的新增催化來擴張估值。
- 這組更適合等 `支撐守穩 + 量能回升` 再加權，而不是現在追價。

### 3. 高 beta / 事件交易

- `IP / RIVER / SIREN / LIT / EDGE`：這組裡的低 RSI 與短線彈性都是真的，但基本面、流動性或治理風險也同樣大。
- `LIT` 是這組裡事件線最完整的一檔；`SIREN` 與 `EDGE` 則是波動最高、最需要嚴格停損的兩檔。
