## 今日分組觀察

### 1. 可優先追蹤

- `BTC / DOT / HYPE / BGB`：這組同時具備清楚敘事與相對可驗證的事件線。
- `BTC` 是宏觀不穩環境下的資金主軸；`DOT` 是基本面強於價格；`HYPE` 有產品擴張與高 beta；`BGB` 則是平台激勵直接綁交易量。

### 2. 防守與平衡組

- `PAXG / XAUt`：`refx` 顯示能源與通膨風險沒有真正退場，黃金代幣仍值得保留。
- `NEXO / KCS`：偏合規、平台經營與現金流邏輯，彈性不如高 beta，但較有中期承接。

### 3. 事件與反彈盤

- `EGLD / DCR / XMR / FIL / ZEN / CAKE / MX`：都有各自催化，但需要更明確的量能或後續事件確認。
- `EGLD / FIL` 比較像等待結構翻多；`XMR / ZEN / HYPE` 則是事件驅動更強、同時波動也更大的那一側。
