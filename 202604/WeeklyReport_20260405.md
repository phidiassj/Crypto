# 每週加密貨幣關注重點 2026-04-05

> 整理區間：2026-03-30 至 2026-04-05 每日追蹤報告
>
> 下週觀察區間：2026-04-06 至 2026-04-12

## 一句話結論

下週主軸仍應以「防守優先、只對有明確基本面延續的標的提高關注」為原則。這週反覆出現的共識很明確：市場恐懼尚未解除、衍生品主導成交、地緣風險未退，但資金並未全面撤出，因此最值得追的不是最熱的高 beta 幣，而是有真實採用、平台效用或避險需求支持的少數標的。

## 下週核心觀察名單

### 1. 防守與資金停泊

- `PAXG` / `XAUt`
  - 這兩檔在本週多次被列為最低 RSI 或最完整的防守型配置，邏輯一致且沒有改變。
  - 若中東風險、總體不確定性與 alt 波動延續，鏈上黃金仍是最合理的風險停泊點。
  - 下週重點不是追價，而是觀察是否持續維持低波動強勢，代表 risk-off 資金還沒離場。

### 2. 中期結構最完整的觀察組

- `ICP`
  - 本週後半段連續被列入優先追蹤，核心理由是瑞士零售支付採用與 `ckBTC` 敘事具體，不只是情緒題材。
  - 下週重點是看市場回檔時能否守住相對強勢，若守得住，代表資金願意為真實採用敘事買單。
- `NEAR`
  - 本週被反覆歸類為 AI agent / execution layer 主線代表，屬於仍有中期敘事想像力的少數標的。
  - 下週要看的是 `1.23` 一帶支撐是否持續有效，以及反彈是否能重新測試 `1.4` 附近壓力。
- `LTC`
  - 本週末新進入高優先清單，理由不是單純老幣輪動，而是券商通路擴張與 LitVM 技術升級同時存在。
  - 若市場轉穩，LTC 比多數老牌幣更有機會得到資金回流。
- `LINK`
  - 雖然後半週不一定天天進入最前排，但整週來看，LINK 是少數同時具備基礎設施採用、整合落地與中期品質的標的。
  - 下週重點是觀察是否重新挑戰 `9.5-10` 壓力區，若突破，代表基礎設施線開始重新獲得市場定價。
- `MNT`
  - 週初表現最完整，靠 Vault AUM、TVL 與 L2 資金流支撐，不是空故事。
  - 後半週雖未持續出現在最強清單，但仍應保留在下週的中期觀察名單。

## 次核心觀察名單

### 1. 平台與基礎設施輪動

- `KCS`
  - 幾乎整週都在清單內，代表資金確實把它當成低 RSI 平台幣觀察，但問題始終沒變：有效用、缺爆發催化。
  - 下週若市場回穩，它有防守性；若市場續弱，它大概率仍是區間整理。
- `CAKE`
  - 週初與週末都被列為可追蹤標的，優勢是通縮與產品線還在更新。
  - 但目前更像平台型反彈，而不是已進入主升段。
- `ENS`
  - 週末新進入高品質觀察組，原因是 ENS 被 AI wallet / agent 基礎設施反覆點名。
  - 如果下週 AI agent 敘事續熱，ENS 可能比一般平台幣更有彈性。
- `HYPE`
  - RWA perpetuals 敘事仍強，但短線已有過熱後整理跡象。
  - 下週適合看支撐，不適合把它當低風險起漲點。

### 2. 有事件線但偏交易型

- `ASTER`
  - 週初仍是少數有主網後採用驗證故事的標的，但後半週存在感下降。
  - 這代表它仍可追蹤，但優先級已低於 `ICP / NEAR / LINK / LTC`。
- `LIT`
  - 幾乎整週都有出現，從跌深反彈到 Telegram / 衍生品流量入口，短線故事很多。
  - 但一致結論也是「偏事件交易，不宜無條件中抱」。
- `QTUM` / `NEO` / `RAY` / `ZEC` / `EDGE`
  - 這組共同特徵是有單點事件或輪動彈性，但缺少整週持續被驗證的高品質主線。
  - 下週適合當交易名單，不適合升格成主配置。

## 下週最好先避開或降低期待

- `GT` / `OKB` / `MX` / `NEXO`
  - 這類平台幣整週都反覆出現「低 RSI 但缺新催化」的同一個問題。
  - 若沒有新的平台級事件，它們更像區間交易標的，而不是值得主動加碼的主線。
- `BSV` / `DASH` / `CVX`
  - 本週多次被歸為反彈型或偏保守觀察，缺點是要嘛缺催化，要嘛歷史包袱重。
  - 除非量價一起轉強，否則下週不該放在前排。
- `SIREN`
  - 週初已明確顯示籌碼集中與高槓桿風險，後續也沒有被重新驗證成高品質機會。
  - 下週若再有異常波動，仍應視為高風險事件交易。

## 本週整合出的市場背景

### 1. 大方向仍偏 risk-off

- `PAXG` / `XAUt` 本週多次居於最有說服力的防守配置，反映資金對避險敘事仍有需求。
- `refx` 在週後半段反覆提到市場恐懼偏高、地緣風險未解、衍生品主導成交，與每日報告結論一致。

### 2. 但資金沒有全面撤退

- BTC ETF 仍有承接、USDT 持續流入交易所待命，表示市場更像「保守但未崩潰」。
- 因此下週策略不應是全面空倉，而是聚焦少數有基本面與採用支撐的標的。

### 3. 觀察重點要從「是否熱門」改成「是否有延續性」

- 本週能反覆留在優先名單的，大多是 `PAXG`、`XAUt`、`ICP`、`LINK`、`MNT`、`NEAR`、`LTC` 這類有一致邏輯支持的標的。
- 只在單日爆出的標的，多半更適合短打，不適合下週當主配置。

## 建議的下週觀察順序

1. `PAXG` / `XAUt`：確認避險資金是否續留。
2. `ICP` / `NEAR` / `LTC` / `LINK`：觀察是否在大盤震盪中維持相對強勢。
3. `MNT` / `ENS` / `CAKE` / `KCS` / `HYPE`：等待關鍵位與量能確認。
4. `ASTER` / `LIT` / `QTUM` / `NEO` / `RAY` / `ZEC` / `EDGE`：只作事件與波段交易名單。
